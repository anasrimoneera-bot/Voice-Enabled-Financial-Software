# 🎙️ 语音记账 · 财务工具

通过**语音**快速录入收支的网页记账应用，可直接部署到 **app.b2bsxlj.com**。

## 功能

- **语音录入**：点击麦克风说出「15号 午餐 35 元」「收到工资 8000 元」「昨天打车 22 块」，自动识别**日期**、**金额**、**项目**、**分类**与**收入/支出**类型。
- **日期识别与排序**：识别语音中的日期（如「15号」「6月15日」「昨天」「前天」），记录按日期**升序排列（1→30 日依次）**；同一日的多笔分别单独显示，允许同日相同金额、相同项目的重复录入。
- **智能分类**：自动将项目归入「餐饮 / 交通 / 购物 / 居住 / 娱乐 / 医疗 / 通讯 / 教育 / 人情 / 工资 / 奖金 / 收入」等分类，并配图标与颜色。
- **统计图表**：近 6 个月收支柱状图 + 按月份的支出分类占比（纯前端绘制，无第三方库）。
- **数据导出**：一键导出 **Excel（.xlsx）** 账单，固定采用标准财务流水表列名：`日期 / 流水号 / 摘要 / 账户对方科目 / 辅助核算 / 对方户名 / 对方账号 / 对方银行 / 收入 / 支出 / 备注 / 类别`。收入/支出按类型分列，备注自动填入「录入方式 + 时间」，类别填入智能分类。XLSX 由内置零依赖写出器生成（`xlsx.js`），无需第三方库或联网。
- **云同步 / 多设备**：可选接入后端，按「同步码」在多台设备间同步账单（含删除同步）。
- **手动录入**：不方便说话时可手动添加（兜底）。
- **收支概览 / 筛选 / 删除**：实时统计收入、支出、结余。
- **本地存储**：数据保存在浏览器 `localStorage`；未配置云同步时无需后端、无需登录。
- **PWA**：可「添加到主屏幕」，像 App 一样使用。

## 技术说明

- 前端为纯静态（HTML / CSS / 原生 JS），**无构建步骤**，可托管在任意静态服务器。
- 语音识别使用浏览器原生 **Web Speech API**（`SpeechRecognition`），支持中文（`zh-CN`）。
  - ⚠️ 该 API 需在 **HTTPS** 下运行才能访问麦克风（你的域名已备案并上线，配好 SSL 即可）。
  - 浏览器兼容：推荐 **Chrome / Edge**（含安卓 Chrome）。Safari 支持有限，iOS 上可使用手动录入。
- 云同步后端为 **零依赖 Node 服务**（仅用内置模块），数据按同步码分文件存储。

## 文件结构

```
index.html              页面结构
style.css               样式
app.js                  语音/日期/金额解析、记录、排序、渲染、导出、同步装配
categories.js           智能分类（关键词 → 分类组 + 图标 + 颜色）
charts.js               统计图表（月度收支、分类占比）
sync.js                 云同步前端逻辑
xlsx.js                 零依赖 XLSX 写出器（导出 Excel）
manifest.webmanifest    PWA 配置
deploy/nginx.conf       Nginx 部署示例（静态站点 + /api 反向代理）
server/server.js        云同步后端（零依赖 Node）
server/package.json     后端启动脚本
```

## 部署到 app.b2bsxlj.com

### 1. 前端（静态站点）

1. **添加 DNS 解析**：在 B2BSXLJ.COM 控制台为子域名 `app` 添加 **A 记录**（指向服务器公网 IP）或 **CNAME 记录**。
2. **上传文件**：将根目录下的 `index.html`、`style.css`、`app.js`、`categories.js`、`charts.js`、`sync.js`、`xlsx.js`、`manifest.webmanifest` 上传到服务器目录，如 `/var/www/app.b2bsxlj.com`。
3. **配置 HTTPS**：为 `app.b2bsxlj.com` 申请 SSL 证书（Let's Encrypt / 阿里云 / 腾讯云均可），语音功能依赖 HTTPS。
4. **配置 Nginx**：参考 `deploy/nginx.conf`，放入 `/etc/nginx/conf.d/` 后执行：
   ```bash
   nginx -t && nginx -s reload
   ```
5. 打开 **https://app.b2bsxlj.com** ，允许麦克风权限即可开始语音记账。

> 也可使用 Vercel / Netlify / 阿里云 OSS / 腾讯云 COS 等静态托管，将自定义域名绑定为 `app.b2bsxlj.com` 并开启 HTTPS（此时云同步需单独部署后端）。

### 2. 云同步后端（可选，需要多设备同步时部署）

```bash
cd server
PORT=3001 node server.js          # 启动后监听 3001 端口
# 建议用 systemd / pm2 守护进程，确保开机自启与崩溃重启
```

`deploy/nginx.conf` 已将 `https://app.b2bsxlj.com/api/` 反向代理到该服务。
在网页底部「☁️ 云同步」中填写：

- **服务器地址**：`https://app.b2bsxlj.com/api`
- **同步码**：自定义口令（如 `myfamily-2026`），同码设备共享同一份数据。

> 安全提示：同步码即访问凭据，请使用不易猜测的字符串。后端按同步码哈希分文件存储，不同步码相互隔离。如需更强的鉴权（账号密码 / Token），可在此基础上扩展。

## 本地预览

```bash
# 前端（http://localhost 下也可使用麦克风）
python3 -m http.server 8080      # 访问 http://localhost:8080

# 后端（可选）
cd server && node server.js
```

## 语音指令示例

| 说出 | 识别结果 |
| --- | --- |
| 15号 午餐 35 元 | 6月15日 · 支出 · 餐饮 · ¥35.00 |
| 昨天打车二十二块 | 昨天 · 支出 · 交通 · ¥22.00 |
| 6月15号买菜 100 元 | 6月15日 · 支出 · 餐饮 · ¥100.00 |
| 收到工资 8000 元 | 今天 · 收入 · 工资 · ¥8000.00 |
| 报销 一千二 | 今天 · 收入 · 收入 · ¥1200.00 |

- 收入关键词：收入、工资、薪水、进账、收到、报销、红包、退款、奖金、利息、分红等；其余默认计为支出。
- 日期支持：`今天 / 昨天 / 前天 / 大前天`、`X号 / X日`、`X月X号`（含中文数字「十五号」「三十号」）；未提及日期则默认当天。
